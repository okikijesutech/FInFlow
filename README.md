# FinFlow - Premium FinTech Dashboard

FinFlow is a modern, professional FinTech dashboard built with React, TypeScript, and Tailwind CSS v4. It features a sleek glassmorphism aesthetic, interactive data visualizations, and a responsive layout designed for high-performance financial management.

## 🚀 Live Demo
[View Live Demo](https://fin-flow-silk.vercel.app/)

## ✨ Features
- **Glassmorphism UI**: High-end aesthetic using the latest Tailwind CSS v4 Oxide engine.
- **Interactive Charts**: Dynamic revenue and spending analytics powered by Recharts.
- **Micro-animations**: Smooth staggered entry and hover effects with Framer Motion.
- **Responsive Layout**: Mobile-first design with a slide-out sidebar and sticky topbar.
- **SOLID Architecture**: Clean, type-safe codebase with custom hooks for data management.

## 🛠️ Tech Stack
- **Framework**: React 19 + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Charts**: Recharts
- **Animations**: Framer Motion

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/okikijesutech/FInFlow.git
   ```

2. **Install dependencies**:
   ```bash
   pnpm install
   ```

3. **Run the development server**:
   ```bash
   pnpm dev
   ```

4. **Build for production**:
   ```bash
   pnpm build
   ```
